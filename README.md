Le projet QCM a été developpé en langage C ce qui a permis sa création, la gestion des fichiers ainsi que l'écution du QCM.
Ce code gère la lecture et l'ecriture des fichiers textes. 



Les Fichiers : 

- main.c : Point d'entrée contenant la boucle principale et le menu.
- qcm.c : Implémentation des fonctionnalités (lecture/écriture fichiers, algorithme du jeu).
- qcm.h : Structures de données (QCM`, `Question) et prototypes des fonctions.
- Fichiers .txt : Fichiers de sauvegarde générés par le programme contenant les questions.
